# Retail Sales & Customer Behaviour Dashboard

This project presents an interactive dashboard built using a retail shopping trends dataset from Kaggle. The dashboard explores customer behaviour, sales performance, and purchasing trends through interactive charts, filters, and key performance indicators (KPIs). The project demonstrates my skills in data cleaning, exploratory data analysis using Microsoft Excel, and presenting insights through an interactive HTML dashboard. The dashboard was developed with AI assistance and customized to accurately reflect the analysis.

## Tools & Technologies

* Microsoft Excel 2016
* HTML Dashboard
* HTML5
* CSS3
* JavaScript
* Chart.js
* Kaggle Retail Shopping Trends Dataset
* GitHub (Project Hosting)
* AI-assisted development (Claude)

## Project Objectives

* Analyze retail sales performance across different product categories.
* Understand customer purchasing behaviour and preferences.
* Identify seasonal sales trends and revenue patterns.
* Explore the impact of promotions and subscription status on sales.
* Present business insights through an interactive dashboard.

## Dataset

* **Source:** Kaggle - Retail Shopping Trends Dataset
* **Records:** Approximately 3,900 customer transactions
* **Format:** CSV
* **Data Preparation:** The raw dataset was cleaned and prepared in Microsoft Excel before being analyzed and used to create the interactive dashboard.

## Key Features

* Interactive dashboard built with HTML, CSS, JavaScript, and Chart.js.
* Dynamic filters for exploring customer and sales data.
* Key Performance Indicators (KPIs) summarizing business performance.
* Visualizations of revenue, customer behaviour, product categories, and seasonal trends.
* Business findings and recommendations based on the data analysis.

## Analysis Workflow

1. Downloaded the retail shopping trends dataset from Kaggle.
2. Cleaned and prepared the data using Microsoft Excel.
3. Performed exploratory data analysis using PivotTables, PivotCharts, and calculated fields.
4. Identified key business insights and trends.
5. Created and customized an AI-assisted interactive HTML dashboard to present business insights from the Excel analysis.
6. Documented the project and prepared it for portfolio presentation.
